# 前端应用

基于 Vue 3 + Vite + Element Plus 的 Web 应用前端。

## 环境要求

- Node.js 16+
- npm 或 yarn

## 安装步骤

1. 安装依赖：
```bash
npm install
```

2. 配置环境变量（可选）：
```bash
# 复制示例文件
copy .env.example .env

# 如果需要修改 API 地址，编辑 .env 文件
```

3. 启动开发服务器：
```bash
npm run dev
```

4. 访问应用：
- http://localhost:5173

## 构建生产版本

```bash
npm run build
```

构建产物在 `dist` 目录中。
